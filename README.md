# Aprendizado-SQL-Server

## Projeto CineSQL
Este banco de dados foi criado para praticar a estrutura básica de tabelas.

O que aprendi aqui:
Criação de tabelas com IDENTITY.

Tipos de dados VARCHAR e INT.

Inserção de dados básicos.

### O que aprendi na parte 3:
* **Relacionamentos (INNER JOIN):** Aprendi a conectar 3 tabelas diferentes para gerar um relatório completo.
* **Integridade de Dados:** Percebi que o SQL depende de IDs exatos. Se eu colocar o ID errado, o sistema 'troca' os atores de filme sem avisar.

### O que aprendi na parte 4:
* No módulo 04, foquei em funções de agregação (SUM, COUNT, AVG) e filtros complexos.
